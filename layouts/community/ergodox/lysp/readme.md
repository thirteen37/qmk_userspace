# LYSP layout

* Mac and Emacs friendly keyboard layout


## Layers

1. QWERTY
2. Fn keys (right hand)
3. Mouse keys (left hand), Numpad (right hand)
4. Gamepad layer (left hand)

See http://configure.ergodox-ez.com/keyboard_layouts/qvmwvy/edit
